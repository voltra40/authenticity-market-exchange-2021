How to run

1) Open Ganache and click quickstart or open your workspace
2) Install the metamask wallet on your browser and connect using your ganache workspace seed phrase (mnemonic)
2) Through the terminal, cd into the project folder
3) Run these commands in the terminal
	cd ame-contract
	truffle compile
	truffle migrate --reset
	cd ../ame-app
	npm install
	npm start
4) Open the application on localhost:3000 in your web browser
5) Connect metamask accounts to application (metamask should automatically activate)
6) Application should now be fully funcitonal
